# Slash Commands Bot

A Discord bot that supports slash commands.

## Installation

Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install
```

## Environment Variables

Copy the .env.example file into .env:

```bash
cp .env.example .env
```

Update the values accordingly.

## Development

Start the bot:

```bash
# yarn
yarn dev

# npm
npm run dev
```

## Production

Build the bot for production:

```bash
# yarn
yarn build

# npm
npm run build
```

Start the bot:

```bash
# yarn
yarn dev

# npm
npm run dev
```
